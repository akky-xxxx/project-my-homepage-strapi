## About
backend for my-homepage

## Requires
- node.js
- yarn
- docker

## Resolve dependencies
```shell
$ yarn
```

## Start as development
```shell
$ yarn dev
```

## Scripts
| script     | description                       |
|------------|-----------------------------------|
| check-all  | run `check-all` of all packages   |
| dev        | run `dev` of all service packages |
| fix        | apply fix task for root files     |
| lint       | check fpr root files              |
